Dicas para Sonoplastas

  • Chegue com antecedência para testar os equipamentos
  • Mantenha uma lista atualizada dos louvores mais utilizados
  • Verifique o funcionamento dos microfones antes do culto
  • Ajuste o volume de acordo com o tamanho do ambiente
  • Tenha sempre pilhas reservas para os equipamentos sem fio
